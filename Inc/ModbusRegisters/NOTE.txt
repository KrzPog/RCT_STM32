Modbus protocol used according to specification:
https://www.modbus.org/docs/Modbus_Application_Protocol_V1_1b3.pdf

Currently enabled functions:
03 (0x03) Read Holding Registers
04 (0x04) Read Input Registers
06 (0x06) Write Single Register
16 (0x10) Write Multiple registers
17 (0x11) Report Server ID

Example CRC Calculator:
https://crccalc.com/?crc=010400000005&method=CRC-16/MODBUS&datatype=hex&outtype=hex
!!! Low Order Byte First (Calculator gives 0x0930 -> append 0x3009 to end of frame)

For testing connection:
REQUEST (hex):  01 11 C0 2C
RESPONSE (hex): 01 11 01 FF 10 13 ( @note sometimes serial viewers don't display last byte -> \r )
